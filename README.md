# Loan-Default-Prediction
End-to-end loan default risk scoring system built on 307,511 real home credit loan applications.  Predicts whether an applicant will default on their loan and outputs a risk tier (A/B/C/D) using  XGBoost and LightGBM. Handles class imbalance with SMOTE, SHAP explainability  for transparent credit decisions, and deployed as a REST API using FastAPI.
